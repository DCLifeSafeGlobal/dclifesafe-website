# DC LifeSafe Global Website

Starter website built with React and Vite.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Upload to GitHub and Vercel

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to Vercel.
4. Click **Import Project**.
5. Select your GitHub repository.
6. Click **Deploy**.

## Main files to edit later

- `src/App.jsx` → page content
- `src/index.css` → design and colors
